# Family Shield 🛡️

**Family Shield** — это программный комплекс (Android + Backend) для защиты пожилых людей от телефонных мошенников в режиме реального времени.

## 🚀 Как это работает

1. **Детекция:** Приложение на телефоне пенсионера через `AccessibilityService` обнаруживает входящий звонок.
2. **Аудио-стриминг:** Сервис автоматически включает громкую связь и начинает стримить аудио через микрофон на сервер по WebSocket (PCM 16 kHz).
3. **Анализ (ASR):** Сервер переводит речь в текст, используя цепочку: **OpenRouter (Whisper-large-v3)** → SaluteSpeech (fallback).
4. **Классификация (LLM):** **GPT-4o-mini** анализирует транскрипт на лету. Если обнаруживаются паттерны мошенничества («безопасный счет», «родственник в беде»), система реагирует мгновенно.
5. **Защита:** 
   - **Пенсионеру:** В динамик проигрывается TTS-предупреждение: *«Внимание! Это может быть мошенник!»*
   - **Родственнику:** Приходит экстренный Push (FCM), сообщение в WebSocket и SMS через **SMSAERO**.

## 🛠 Стек технологий

### Backend (Python)
- **FastAPI 0.115** — высокопроизводительный API и WebSocket.
- **SQLAlchemy 2.0 + SQLite** — хранение истории звонков и настроек.
- **OpenRouter API** — основной шлюз для ASR (Whisper) и LLM (GPT-4o-mini).
- **SMSAERO** — шлюз для экстренных SMS-уведомлений.
- **Firebase Admin SDK** — отправка Push-уведомлений.

### Android (Kotlin)
- **Jetpack Compose** — современный UI в стиле Material 3.
- **AccessibilityService** — автоматизация управления звонком (громкая связь).
- **Foreground Service** — стабильная работа и запись аудио в фоновом режиме.
- **OkHttp WebSocket** — real-time передача аудио и получение команд от сервера.

## ⚙️ Быстрый старт

### Backend
1. Установите зависимости: `pip install -r requirements.txt`
2. Создайте `.env` на основе `.env.example` и добавьте свои API-ключи.
3. Запустите сервер: `uvicorn app.main:app --reload`
4. Проверьте статус: `GET http://localhost:8000/api/health`

### Android
1. Откройте папку `android/` в Android Studio.
2. Добавьте свой `google-services.json` в `app/`.
3. В `local.properties` укажите `wsBaseUrl` вашего сервера.
4. Соберите и установите APK.

## 🛡 Безопасность и PII
Приложение фильтрует персональные данные (номера карт, телефоны) на стороне сервера перед отправкой в LLM. Аудиозаписи не хранятся на сервере в целях приватности — сохраняются только текстовые транскрипты для родственников.
